# PodcastCogimix
Simple python (draft) app to serve podcast to Cogimix.com

Dependencies :

	python 2.7
	bottle==0.12.8
	bottle-mongo==0.2.4
	pymongo==2.8   
	requests==2.5.3

Install :

pip install -r requirements.txt

Usage:

python podcast_cogimix.py

wget -qO- localhost:8000/refresh &> /dev/null
